312 npy/npz files containing object arrays. Values other than 'latin1', 

313 'ASCII', and 'bytes' are not allowed, as they can corrupt numerical 

314 data. Default: 'ASCII' 

315

320 of NpzFile class must be closed to avoid leaking file descriptors. 

321

337 - If the file contains pickle data, then whatever object is stored 

338 in the pickle is returned.

340 - If the file is a ``.npz`` file, then a dictionary-like object is 

341 returned, containing ``{filename: array}`` key-value pairs, one for

368 array([[1, 2, 3],

372 >>> data.close()

375 directly from disk:

742 def floatconv(x): 

743 x.lower() 

744 if '0x' in x: 

745 return float.fromhex(x) 

746 return float(x)

748 typ = dtype.type 

749 if issubclass(typ, np.bool_):

773def loadtxt(fname, dtype=float, comments='#', delimiter=None, 

774 converters=None, skiprows=0, usecols=None, unpack=False, 

775 ndmin=0, encoding='bytes'): 

776 """ 

777 Load data from a text file. 

778 

779 Each row in the text file must have the same number of values. 

780 

781 Parameters 

782 ---------- 

783 fname : file, str, or pathlib.Path 

784 File, filename, or generator to read. If the filename extension is 

785 ``.gz`` or ``.bz2``, the file is first decompressed. Note that 

786 generators should return byte strings for Python 3k. 

787 dtype : data-type, optional 

788 Data-type of the resulting array; default: float. If this is a 

789 structured data-type, the resulting array will be 1-dimensional, and 

790 each row will be interpreted as an element of the array. In this 

791 case, the number of columns used must match the number of fields in 

792 the data-type. 

793 comments : str or sequence of str, optional 

794 The characters or list of characters used to indicate the start of a 

795 comment. None implies no comments. For backwards compatibility, byte 

796 strings will be decoded as 'latin1'. The default is '#'. 

797 delimiter : str, optional 

798 The string used to separate values. For backwards compatibility, byte 

799 strings will be decoded as 'latin1'. The default is whitespace. 

800 converters : dict, optional 

801 A dictionary mapping column number to a function that will parse the 

802 column string into the desired value. E.g., if column 0 is a date 

803 string: ``converters = {0: datestr2num}``. Converters can also be 

804 used to provide a default value for missing data (but see also 

805 `genfromtxt`): ``converters = {3: lambda s: float(s.strip() or 0)}``. 

806 Default: None. 

807 skiprows : int, optional 

808 Skip the first `skiprows` lines; default: 0. 

809 usecols : int or sequence, optional 

810 Which columns to read, with 0 being the first. For example, 

811 ``usecols = (1,4,5)`` will extract the 2nd, 5th and 6th columns. 

812 The default, None, results in all columns being read. 

813 

814 .. versionchanged:: 1.11.0 

815 When a single column has to be read it is possible to use 

816 an integer instead of a tuple. E.g ``usecols = 3`` reads the 

817 fourth column the same way as ``usecols = (3,)`` would. 

818 unpack : bool, optional 

819 If True, the returned array is transposed, so that arguments may be 

820 unpacked using ``x, y, z = loadtxt(...)``. When used with a structured 

821 data-type, arrays are returned for each field. Default is False. 

822 ndmin : int, optional 

823 The returned array will have at least `ndmin` dimensions. 

824 Otherwise mono-dimensional axes will be squeezed. 

825 Legal values: 0 (default), 1 or 2. 

826 

827 .. versionadded:: 1.6.0 

828 encoding : str, optional 

829 Encoding used to decode the inputfile. Does not apply to input streams. 

830 The special value 'bytes' enables backward compatibility workarounds 

831 that ensures you receive byte arrays as results if possible and passes 

832 'latin1' encoded strings to converters. Override this value to receive 

833 unicode arrays and pass strings as input to converters. If set to None 

834 the system default is used. The default value is 'bytes'. 

835 

836 .. versionadded:: 1.14.0 

837 

838 Returns 

839 ------- 

840 out : ndarray 

841 Data read from the text file. 

842 

843 See Also 

844 -------- 

845 load, fromstring, fromregex 

846 genfromtxt : Load data with missing values handled as specified. 

847 scipy.io.loadmat : reads MATLAB data files 

848 

849 Notes 

850 ----- 

851 This function aims to be a fast reader for simply formatted files. The 

852 `genfromtxt` function provides more sophisticated handling of, e.g., 

853 lines with missing values. 

854 

855 .. versionadded:: 1.10.0 

856 

857 The strings produced by the Python float.hex method can be used as 

858 input for floats. 

859 

860 Examples 

861 -------- 

862 >>> from io import StringIO # StringIO behaves like a file object 

863 >>> c = StringIO(u"0 1\\n2 3") 

864 >>> np.loadtxt(c) 

865 array([[ 0., 1.], 

866 [ 2., 3.]]) 

867 

868 >>> d = StringIO(u"M 21 72\\nF 35 58") 

869 >>> np.loadtxt(d, dtype={'names': ('gender', 'age', 'weight'), 

870 ... 'formats': ('S1', 'i4', 'f4')}) 

871 array([('M', 21, 72.0), ('F', 35, 58.0)], 

872 dtype=[('gender', '|S1'), ('age', '<i4'), ('weight', '<f4')]) 

873 

874 >>> c = StringIO(u"1,0,2\\n3,0,4") 

875 >>> x, y = np.loadtxt(c, delimiter=',', usecols=(0, 2), unpack=True) 

876 >>> x 

877 array([ 1., 3.]) 

878 >>> y 

879 array([ 2., 4.]) 

880 

881 """ 

882 # Type conversions for Py3 convenience 

883 if comments is not None: 

884 if isinstance(comments, (basestring, bytes)): 

885 comments = [comments] 

886 comments = [_decode_line(x) for x in comments] 

887 # Compile regex for comments beforehand 

888 comments = (re.escape(comment) for comment in comments) 

889 regex_comments = re.compile('|'.join(comments)) 

890 

891 if delimiter is not None: 

892 delimiter = _decode_line(delimiter) 

893 

894 user_converters = converters 

895 

896 if encoding == 'bytes': 

897 encoding = None 

898 byte_converters = True 

899 else: 

900 byte_converters = False 

901 

902 if usecols is not None: 

903 # Allow usecols to be a single int or a sequence of ints 

904 try: 

905 usecols_as_list = list(usecols) 

906 except TypeError: 

907 usecols_as_list = [usecols] 

908 for col_idx in usecols_as_list: 

909 try: 

910 opindex(col_idx) 

911 except TypeError as e: 

912 e.args = ( 

913 "usecols must be an int or a sequence of ints but " 

914 "it contains at least one element of type %s" % 

915 type(col_idx), 

916 )

919 usecols = usecols_as_list 

920 

921 fown = False 

922 try: 

923 if is_pathlib_path(fname): 

924 fname = str(fname)

926 fh = np.lib._datasource.open(fname, 'rt', encoding=encoding) 

927 fencoding = getattr(fh, 'encoding', 'latin1') 

928 fh = iter(fh) 

929 fown = True 

930 else: 

931 fh = iter(fname) 

932 fencoding = getattr(fname, 'encoding', 'latin1') 

933 except TypeError: 

934 raise ValueError('fname must be a string, file handle, or generator')

937 if encoding is not None: 

938 fencoding = encoding 

939 # we must assume local encoding 

940 # TODO emit portability warning? 

941 elif fencoding is None: 

942 import locale 

943 fencoding = locale.getpreferredencoding()

945 # not to be confused with the flatten_dtype we import... 

946 def flatten_dtype_internal(dt): 

947 """Unpack a structured data-type, and produce re-packing info.""" 

948 if dt.names is None: 

949 # If the dtype is flattened, return. 

950 # If the dtype has a shape, the dtype occurs 

951 # in the list more than once. 

952 shape = dt.shape 

953 if len(shape) == 0: 

954 return ([dt.base], None) 

955 else: 

956 packing = [(shape[-1], list)] 

957 if len(shape) > 1: 

958 for dim in dt.shape[-2::-1]: 

959 packing = [(dim*packing[0][0], packing*dim)] 

960 return ([dt.base] * int(np.prod(dt.shape)), packing) 

961 else: 

962 types = [] 

963 packing = [] 

964 for field in dt.names: 

965 tp, bytes = dt.fields[field] 

966 flat_dt, flat_packing = flatten_dtype_internal(tp) 

967 types.extend(flat_dt) 

968 # Avoid extra nesting for subarrays 

969 if tp.ndim > 0: 

970 packing.extend(flat_packing) 

971 else: 

972 packing.append((len(flat_dt), flat_packing)) 

973 return (types, packing)

975 def pack_items(items, packing): 

976 """Pack items into nested lists based on re-packing info.""" 

977 if packing is None: 

978 return items[0] 

979 elif packing is tuple: 

980 return tuple(items) 

981 elif packing is list: 

982 return list(items) 

983 else: 

984 start = 0 

985 ret = [] 

986 for length, subpacking in packing: 

987 ret.append(pack_items(items[start:start+length], subpacking)) 

988 start += length 

989 return tuple(ret)

1003 def read_data(chunk_size): 

1004 """Parse each line, including the first. 

1005 

1006 The file read, `fh`, is a global defined above. 

1007 

1008 Parameters 

1009 ---------- 

1010 chunk_size : int 

1011 At most `chunk_size` lines are read at a time, with iteration 

1012 until all lines are read. 

1013 

1014 """ 

1015 X = [] 

1016 for i, line in enumerate(itertools.chain([first_line], fh)): 

1017 vals = split_line(line) 

1018 if len(vals) == 0: 

1019 continue 

1020 if usecols: 

1021 vals = [vals[j] for j in usecols] 

1022 if len(vals) != N: 

1023 line_num = i + skiprows + 1 

1024 raise ValueError("Wrong number of columns at line %d" 

1025 % line_num) 

1026 

1027 # Convert each value according to its column and store 

1028 items = [conv(val) for (conv, val) in zip(converters, vals)] 

1029 

1030 # Then pack it according to the dtype's nesting 

1031 items = pack_items(items, packing) 

1032 X.append(items) 

1033 if len(X) > chunk_size: 

1034 yield X 

1035 X = [] 

1036 if X: 

1037 yield X 

1038 

1039 try: 

1040 # Make sure we're dealing with a proper dtype 

1041 dtype = np.dtype(dtype) 

1042 defconv = _getconv(dtype) 

1043 

1044 # Skip the first `skiprows` lines 

1045 for i in range(skiprows): 

1046 next(fh) 

1047 

1048 # Read until we find a line with some values, and use 

1049 # it to estimate the number of columns, N. 

1050 first_vals = None 

1051 try: 

1052 while not first_vals: 

1053 first_line = next(fh) 

1054 first_vals = split_line(first_line) 

1055 except StopIteration: 

1056 # End of lines reached 

1057 first_line = '' 

1058 first_vals = [] 

1059 warnings.warn('loadtxt: Empty input file: "%s"' % fname, stacklevel=2) 

1060 N = len(usecols or first_vals) 

1061 

1062 dtype_types, packing = flatten_dtype_internal(dtype) 

1063 if len(dtype_types) > 1: 

1064 # We're dealing with a structured array, each field of 

1065 # the dtype matches a column 

1066 converters = [_getconv(dt) for dt in dtype_types] 

1067 else: 

1068 # All fields have the same dtype 

1069 converters = [defconv for i in range(N)] 

1070 if N > 1: 

1071 packing = [(N, tuple)] 

1072 

1073 # By preference, use the converters specified by the user 

1074 for i, conv in (user_converters or {}).items(): 

1075 if usecols: 

1076 try: 

1077 i = usecols.index(i) 

1078 except ValueError: 

1079 # Unused converter specified 

1080 continue 

1081 if byte_converters: 

1082 # converters may use decode to workaround numpy's old behaviour, 

1083 # so encode the string again before passing to the user converter 

1084 def tobytes_first(x, conv): 

1085 if type(x) is bytes: 

1086 return conv(x) 

1087 return conv(x.encode("latin1")) 

1088 import functools 

1089 converters[i] = functools.partial(tobytes_first, conv=conv) 

1090 else: 

1091 converters[i] = conv 

1092 

1093 converters = [conv if conv is not bytes else 

1094 lambda x: x.encode(fencoding) for conv in converters] 

1095 

1096 # read data in chunks and fill it into an array via resize 

1097 # over-allocating and shrinking the array later may be faster but is 

1098 # probably not relevant compared to the cost of actually reading and 

1099 # converting the data 

1100 X = None 

1101 for x in read_data(_loadtxt_chunksize): 

1102 if X is None: 

1103 X = np.array(x, dtype) 

1104 else: 

1105 nshape = list(X.shape) 

1106 pos = nshape[0] 

1107 nshape[0] += len(x) 

1108 X.resize(nshape, refcheck=False) 

1109 X[pos:, ...] = x 

1110 finally: 

1111 if fown: 

1112 fh.close() 

1113 # recursive closures have a cyclic reference to themselves, which 

1114 # requires gc to collect (gh-10620). To avoid this problem, for 

1115 # performance and PyPy friendliness, we break the cycle: 

1116 flatten_dtype_internal = None 

1117 pack_items = None 

1118 

1119 if X is None: 

1120 X = np.array([], dtype) 

1121 

1122 # Multicolumn data are returned with shape (1, N, M), i.e. 

1123 # (1, 1, M) for a single row - remove the singleton dimension there 

1124 if X.ndim == 3 and X.shape[:2] == (1, 1): 

1125 X.shape = (1, -1) 

1126 

1127 # Verify that the array has at least dimensions `ndmin`. 

1128 # Check correctness of the values of `ndmin` 

1129 if ndmin not in [0, 1, 2]: 

1130 raise ValueError('Illegal value of ndmin keyword: %s' % ndmin) 

1131 # Tweak the size and shape of the arrays - remove extraneous dimensions 

1132 if X.ndim > ndmin: 

1133 X = np.squeeze(X) 

1134 # and ensure we have the minimum number of dimensions asked for 

1135 # - has to be in this order for the odd case ndmin=1, X.squeeze().ndim=0 

1136 if X.ndim < ndmin: 

1137 if ndmin == 1: 

1138 X = np.atleast_1d(X) 

1139 elif ndmin == 2: 

1140 X = np.atleast_2d(X).T 

1141 

1142 if unpack: 

1143 if len(dtype_types) > 1: 

1144 # For structured arrays, return an array for each field. 

1145 return [X[field] for field in dtype.names] 

1146 else: 

1147 return X.T 

1148 else: 

1149 return X

1405

1409

1411 ---------- 

1412 file : str or file 

1413 File name or file object to read. 

1414 regexp : str or regexp 

1415 Regular expression used to parse the file.

1417 dtype : dtype or list of dtypes 

1418 Dtype for the structured array. 

1419 encoding : str, optional 

1420 Encoding used to decode the inputfile. Does not apply to input streams. 

1421 

1422 .. versionadded:: 1.14.0 

1423 

1424 Returns 

1425 ------- 

1426 output : ndarray 

1427 The output array, containing the part of the content of `file` that 

1428 was matched by `regexp`. `output` is always a structured array. 

1429 

1430 Raises 

1431 ------ 

1432 TypeError 

1433 When `dtype` is not a valid dtype for a structured array. 

1434 

1435 See Also 

1436 -------- 

1437 fromstring, loadtxt 

1438 

1439 Notes 

1440 ----- 

1441 Dtypes for structured arrays can be specified in several forms, but all 

1442 forms specify at least the data type and field name. For details see 

1443 `doc.structured_arrays`. 

1444 

1445 Examples 

1446 -------- 

1447 >>> f = open('test.dat', 'w') 

1448 >>> f.write("1312 foo\\n1534 bar\\n444 qux") 

1449 >>> f.close() 

1450 

1451 >>> regexp = r"(\\d+)\\s+(...)" # match [digits, whitespace, anything] 

1452 >>> output = np.fromregex('test.dat', regexp, 

1453 ... [('num', np.int64), ('key', 'S3')]) 

1454 >>> output 

1455 array([(1312L, 'foo'), (1534L, 'bar'), (444L, 'qux')], 

1456 dtype=[('num', '<i8'), ('key', '|S3')]) 

1457 >>> output['num'] 

1458 array([1312, 1534, 444], dtype=int64) 

1459 

1460 """ 

1461 own_fh = False 

1462 if not hasattr(file, "read"): 

1463 file = np.lib._datasource.open(file, 'rt', encoding=encoding) 

1464 own_fh = True 

1465 

1466 try: 

1467 if not isinstance(dtype, np.dtype): 

1468 dtype = np.dtype(dtype) 

1469 

1470 content = file.read() 

1471 if isinstance(content, bytes) and isinstance(regexp, np.unicode): 

1472 regexp = asbytes(regexp) 

1473 elif isinstance(content, np.unicode) and isinstance(regexp, bytes): 

1474 regexp = asstr(regexp) 

1475 

1476 if not hasattr(regexp, 'match'): 

1477 regexp = re.compile(regexp) 

1478 seq = regexp.findall(content) 

1479 if seq and not isinstance(seq[0], tuple): 

1480 # Only one group is in the regexp. 

1481 # Create the new array as a single data-type and then 

1482 # re-interpret as a single-field structured array. 

1483 newdtype = np.dtype(dtype[dtype.names[0]]) 

1484 output = np.array(seq, dtype=newdtype) 

1485 output.dtype = dtype 

1486 else: 

1487 output = np.array(seq, dtype=dtype) 

1488 

1489 return output 

1490 finally: 

1491 if own_fh: 

1492 file.close()
