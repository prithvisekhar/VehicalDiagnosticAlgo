117 """ 

118 Test element-wise for positive infinity, return result as bool array.

120 Parameters

122 x : array_like 

123 The input array.

125 A boolean array with the same shape as `x` to store the result. 

126 

127 Returns 

128 ------- 

129 out : ndarray 

130 A boolean array with the same dimensions as the input. 

131 If second argument is not supplied then a boolean array is returned 

132 with values True where the corresponding element of the input is

134 not positive infinity. 

135 

136 If a second argument is supplied the result is stored there. If the 

137 type of that array is a numeric type the result is represented as zeros 

138 and ones, if the type is boolean then as False and True. 

139 The return value `out` is then a reference to that array. 

140 

141 See Also 

142 -------- 

143 isinf, isneginf, isfinite, isnan 

144 

145 Notes 

146 ----- 

147 NumPy uses the IEEE Standard for Binary Floating-Point for Arithmetic 

148 (IEEE 754). 

149 

150 Errors result if the second argument is also supplied when x is a scalar 

151 input, if first and second arguments have different shapes, or if the 

152 first argument has complex values 

153 

154 Examples 

155 -------- 

156 >>> np.isposinf(np.PINF) 

157 array(True, dtype=bool) 

158 >>> np.isposinf(np.inf) 

159 array(True, dtype=bool) 

160 >>> np.isposinf(np.NINF) 

161 array(False, dtype=bool) 

162 >>> np.isposinf([-np.inf, 0., np.inf]) 

163 array([False, False, True]) 

164

170 array([0, 0, 1])

178 'because it would be ambiguous.') 

179 else:
