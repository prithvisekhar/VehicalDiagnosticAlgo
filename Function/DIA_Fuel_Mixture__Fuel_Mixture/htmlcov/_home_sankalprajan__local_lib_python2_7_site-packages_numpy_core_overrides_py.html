71def verify_matching_signatures(implementation, dispatcher): 

72 """Verify that a dispatcher function has the right signature.""" 

73 implementation_spec = ArgSpec(*getargspec(implementation)) 

74 dispatcher_spec = ArgSpec(*getargspec(dispatcher)) 

75 

76 if (implementation_spec.args != dispatcher_spec.args or 

77 implementation_spec.varargs != dispatcher_spec.varargs or 

78 implementation_spec.keywords != dispatcher_spec.keywords or 

79 (bool(implementation_spec.defaults) != 

80 bool(dispatcher_spec.defaults)) or 

81 (implementation_spec.defaults is not None and 

82 len(implementation_spec.defaults) != 

83 len(dispatcher_spec.defaults))): 

84 raise RuntimeError('implementation and dispatcher for %s have ' 

85 'different function signatures' % implementation) 

86 

87 if implementation_spec.defaults is not None: 

88 if dispatcher_spec.defaults != (None,) * len(dispatcher_spec.defaults): 

89 raise RuntimeError('dispatcher functions can only use None for ' 

90 'default argument values')

114 

115 See NEP-18 for example usage. 

116 

117 Parameters 

118 ---------- 

119 dispatcher : callable

122 array-like arguments to check for ``__array_function__``.

124 __module__ attribute to set on new function, e.g., ``module='numpy'``. 

125 By default, module is copied from the decorated function.

133 ``func(*args, **kwargs)``. 

134 docs_from_dispatcher : bool, optional 

135 If True, copy docs from the dispatcher function onto the dispatched 

136 function, rather than from the implementation. This is useful for

139 Returns
